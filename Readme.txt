The task list application consists of two systems:

1) Authentication system:

Registration: Through which the user can register in the application.
Login: Through which the user can enter the application.
Logout: Through which the user can exit the application.

So that the user is prevented from adding, modifying or deleting any task without being logged in.

2) CRUD operations:

C: Create tasks.
R: Retrieve all tasks by logged in user.
U: Update tasks.
D: Delete tasks.

Mark tasks as completed => Put a line through the completed task.

Search: User can search among all tasks.

Handle navigation bar :
Show sign up and login buttons to user who is not signed in or not logged in.
Show Logout button to user who is signed in .
